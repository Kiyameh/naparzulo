Iniciar servidor: 

    >npm run dev


Estructura de proyecto: 

src
    controlers: Funciones
    middlewares: Control de acceso a paginas sin login
    models: Modelos de datos
    routes: Rutas del servidor
    schemas: Validaciones en back-end
    libs: Código reutilizable

    app.js: Código de express. 
    config.js: Configuraciones globales
    db.js: Conexión base de datos
    index.js: Iniciación y exportación
